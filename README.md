## 📖 Overview

This repository contains the complete code implementation for the paper:

**"A Generalized Reflection-Count-Based Geometrical Optics Framework for Attenuation Analysis in Multimode Waveguides"**

The framework introduces a novel two-parameter attenuation model that decomposes optical loss into:
- **Path-length-dependent absorption** (\( \alpha_{abs} \))
- **Reflection-count-dependent surface scattering** (\( \alpha_{sc} \))

## 🚀 Features

- ✅ Analytical calculation of internal reflection count for planar and cylindrical waveguides
- ✅ Two-parameter attenuation model with physical decomposition
- ✅ Optimal launch angle optimization
- ✅ Mode-dependent loss (MDL) analysis
- ✅ Bending loss extension
- ✅ Wavelength-dependent analysis
- ✅ Monte Carlo ray-tracing simulation
- ✅ Comprehensive figure generation for publication
